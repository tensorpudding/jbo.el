# jbo.el

Emacs plugin to the Lojban dictionary jbo (http://github.com/dag/jbo)

## Install Requirements:

* jbo 
* A recent version of GNU Emacs

## How to install

* Add the following lines to .emacs

  (add-to-list 'load-path "path/to/directory/with/jbo.el")
  (require 'jbo)

## How to use

* M-x jbo-lookup
